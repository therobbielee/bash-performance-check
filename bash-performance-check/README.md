# Performance checker
A performance checker written in bash

Checks:
- CPU %
- System load
- RAM Usage %
- Disk usage %
- RAID status
- ZFS pool status

Notifications:
- ntfy
- Pushover


#Usage
Download the script and .env, change the variables in .env to your liking, and probably set it up as a cron job on whatever bash-friendly machines you want to monitor
